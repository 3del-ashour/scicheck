- Conflicts of interest explicitly mentioned (industry_funded lowers score)
- Retraction or correction notices (retracted → score ≤ 0.05)
- Recency relative to the scientific field (very old studies may be superseded)

  0.70–0.89 → Solid peer-reviewed study in reputable journal
  0.50–0.69 → Reasonable study but limited scope, preprint, or minor COI concern
  0.30–0.49 → Low-quality study, highly speculative, or unknown venue
  0.00–0.29 → Blog, opinion piece, retracted, or clear misinformation source

Allowed flag values (use only from this list, can combine):
  "peer_reviewed", "preprint", "meta_analysis", "rct", "cohort", "case_report",
  "industry_funded", "retracted", "small_sample", "old_study", "high_quality_journal"

Output STRICT JSON only — no markdown, no explanation, no code fences:

  ]
}

Rules:
